# Electronic Address

## Definition
An electronic address assigned to an [Agent](../entities/Organisation_Unit.md). 

## Note
An electronic address might be [Email address](../datatypes/Email_Address.md), [Telephone number](../datatypes/Phone_Number.md), [Website URL](../datatypes/Website_Address.md).
Moreover, additional subtypes of Electronic Address can be created in CERIF modules.
